# Product

Spatia is a native macOS disk space visualizer.

## Positioning

Spatia is not a Mac cleaner. It is a file space map that helps users understand where disk space is going.

## Product Principles

- Show first, do not clean first.
- Default to safe actions: reveal in Finder, copy path, inspect, preview.
- Deletion must be explicit, reversible, and limited to Move to Trash.
- Do not scan private locations until the user chooses a source.
- Explain inaccessible locations without pressuring users into Full Disk Access.
- Keep all file names and paths local.

## Target User

The primary user is a normal Mac user who wants to find large files or folders without learning filesystem internals.

## Non-Goals

- automatic cleanup recommendations
- system optimization claims
- permanent deletion
- background indexing
- telemetry
- cloud sync
- App Store sandbox support in the first phase
